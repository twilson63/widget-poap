# Permapages STAMP Widget 

## Usage

```sh
<script src="https://arweave.net/XXXXXXXXXX"></script>
<div id="passport"></div>
```

> NOTE: This widget depends on the Connector Widget - https://github.com/twilson63/widget-connector

## Build Widget

```sh
npm run build
```

## Run Example Server

```sh
npm run example
```

## Deploy to Arweave

```sh
npm run deploy [path to wallet.json]
```
